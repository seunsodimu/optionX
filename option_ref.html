Update History
8/1/2017
Updated Navigation Definition by Adding: Any system that requires a cellphone or OnStar service to operate is NOT considered
navigation.
Updated Sunroof by Adding: Jeep My Sky roof should be optioned as a sunroof.
4/1/2016
Added Bluetooth Definition: The ability to pair a Bluetooth enabled phone wirelessly with a vehicle that allows a user to make
hands- free phone calls. Note: It is best not to option this feature if you cannot confirm it exists.
Added Adjustable Suspension Definition: A system that allows the driver to manually raise and lower the suspension to alter the
overall ride height of the vehicle. Note: Driver selected suspension modes (i.e. race, comfort, sport, off-road) are not considered
adjustable suspension.
Added Turbo Diesel Definition: Diesel engine with a turbo. An exhaust driven compressor that improves engine performance.
Removed Satellite TV Ready: We no longer option this as a company.
Updated Rear Spoiler Definition to read: Deck lid/trunk/hatch mounted spoiler or wing that spans the full length of the deck
lid/trunk/hatch. Should not be used for SUVs, Vans or Trucks.
Updated Quad Seat Definition to read: Second row seats that are separated from each other. Should only be used for SUVs, or
Vans.
Updated Shaker 500 Sound to Shaker Auto: Premium Sound across multiple makes/models. Commonly found in Ford. (Refers only
to Factory Equipped). Hint: Also used for Shaker 500
2
Updated Fold Away Third Row Definition by Adding: This option should not be used for vehicles whose rear seats fold up and
away (i.e. Lexus LX570).
Updated Leatherette Seats Definition by Adding: Softex, MB-Tex, and V-Tex are all manufacture names for leatherette.
Updated Blind Spot Definition by Adding: Honda Lane Watch should not be optioned as Blind Spot.
Updated Panoramic Sunroof by Adding: A Jeep Liberty with a cloth SkySlider roof should also be optioned as Panoramic Sunroof.
